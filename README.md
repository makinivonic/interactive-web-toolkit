# Interactive Web Toolkit

A responsive and interactive web application that combines multiple utility tools including a quiz system and a calculator. Built using core web technologies with a focus on clean UI design, modular structure, and smooth user experience.

---

## 🚀 Features

- 🧠 Interactive Quiz System with dynamic question rendering
- ➗ Functional Calculator with basic arithmetic operations
- 🎨 Clean and responsive UI design
- ⚡ Fast performance with vanilla JavaScript (no frameworks)
- 📱 Mobile-friendly layout

---

## 🛠️ Built With

- HTML5 – Structure and layout
- CSS3 – Styling and responsive design
- JavaScript (Vanilla) – Logic and interactivity
